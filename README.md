# Data Analysis Projects

Небольшие проекты по анализу данных и Python.

## О себе

Начинающий аналитик данных с техническим образованием и опытом анализа технических метрик.  
Работал с данными мониторинга, анализировал инциденты и автоматизировал обработку данных с помощью Python.

Интересуюсь:
- анализом данных
- продуктовой аналитикой
- автоматизацией обработки данных
- разработкой аналитических скриптов

## Используемые технологии

- Python
- SQL
- Pandas
- Data Analysis

## Проекты

### 1. User Data Analysis
Анализ пользовательских данных и поиск активных пользователей.

### 2. Fraud Detection Script
Простой алгоритм обнаружения подозрительных транзакций.

### 3. Prime Numbers Algorithm
Алгоритм поиска простых чисел с использованием решета Эратосфена.
